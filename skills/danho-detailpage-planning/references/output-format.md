# Output Format (기획서 양식)

기획 완료 후 출력할 기획서 템플릿입니다.

> ⚠️ **v3 변경점 (2026-05)**: 이 문서의 하단부에 있는 "이미지 배치" 관련 부분은 v2 호환용으로 남아있습니다. v3에서는:
> - **이미지 섹션을 별도로 분리하지 않습니다** (예: `### 03. image-empathy` 같은 섹션 만들지 않기)
> - 각 텍스트 섹션 끝에 **이미지 변환 후보** 한 줄만 추가 (`REPLACE_CANDIDATE` / `SUPPORT_CANDIDATE` / `NONE`)
> - 실제 이미지의 위치·종류·텍스트 포함 여부는 **코딩 1차 완료 후 image-plan.md에서 확정**
> - 자세한 v3 마킹 규칙: [SKILL.md의 Step 5](../SKILL.md#step-5-이미지-변환-후보-마킹-이미지-분리-금지)
> - 중복 방지 의사결정: [../../danho-detailpage-coding/references/text-image-deduplication.md](../../danho-detailpage-coding/references/text-image-deduplication.md)

## 기획서 템플릿

```markdown
# 상세페이지 기획서: {제품명}

## 기본 정보
- 브랜드: {브랜드명}
- 제품명: {제품명}
- 카테고리: {장르}
- 가격: {가격}
- 톤앤매너: {tone}
- 권장 컬러테마: {Dark Luxury / Clean Minimal / Warm Natural 등}
- 권장 폰트: 헤드라인 {font}, 본문 {font}

## 제품 호칭 (본문 전체 일관 적용 — 필수) ⭐

- **정식 풀네임**: {예: 알파 식기세척기 초벌 전용 실리콘 수세미} — hero/CTA/첫 등장
- **정식 단축형**: {예: 알파 실리콘 수세미} — 본문 paragraph·카드·alt 일상 호칭
- **브랜드 단독**: {예: ALPHA / 알파} — 푸터·로고 그래픽 전용, 본문 금지

> ⚠️ 본문에서 줄여서 "{브랜드 단독}만" 쓰는 표기 금지. 일관성 검증 grep:
> `grep "{브랜드}" build/*.html | grep -v "{풀네임 또는 단축형 핵심 키워드}"` 출력 0이어야 합격.
> 상세: [product-naming-consistency.md](product-naming-consistency.md)

## 섹션 구성 (장르 기반)
적용 장르: {장르명}
총 섹션 수: {N}개 (권장 최소 12개)
섹션 순서: {1. HOOK → 2. HERO → ... (해당 장르 권장 순서)}

### 단계 분할 적용
| 설득 포인트 | 분할 단계 | 섹션명 |
|-------------|-----------|--------|
| {FEATURES} | 3단계 | 문제인식, 기능설명, 효과 |
| {MATERIALS} | 2단계 | 소재소개, 소재효과 |
| ... | ... | ... |

---

## 섹션별 콘텐츠

> ⚠️ **필수**: 각 섹션마다 이미지 구성을 반드시 작성하세요.
> 이미지가 필요 없는 섹션(HOOK, SPECS, FOOTER 등)은 "이미지 없음 (텍스트 중심)"으로 명시합니다.
> **🆕 이미지 위치**: 각 이미지의 배치 위치(텍스트 전/후/사이/교차)를 명시합니다.

### 1. HOOK
**타입**: {질문형/선언형/고통형}
**카피**:
> {hook copy}

**이미지 구성**: 이미지 없음 (텍스트 중심 섹션)

---

### 2. HERO
**제품명 표기**: {display name}
**핵심 가치**: {one-line value proposition}
**태그라인**: {tagline}

**이미지 배치** (필수 2개) - 📍 위치: 텍스트 후
- 메인: {제품 히어로샷 - 구체적 설명: 배경, 조명, 앵글 등}
- 보조1: {제품 다각도 또는 분위기 연출 - 구체적 설명}

---

### 3. SOCIAL PROOF (해당 시)
**수치 데이터**: {판매량, 재구매율, 평점 등}

**이미지 배치** (권장 1~2개) - 📍 위치: 텍스트 후
- 메인: {인증 뱃지, 수상 로고 등}
- 보조1: {미디어 노출, 수상 내역 이미지}

---

### 4. REVIEWS
**리뷰 1**: {리뷰 내용 - 작성자 정보}
**리뷰 2**: {리뷰 내용 - 작성자 정보}
**리뷰 3**: {리뷰 내용 - 작성자 정보}

**이미지 배치** (필수 2~3개) - 📍 위치: **리뷰 사이 (텍스트 사이)**
- 메인: {리뷰 캡처 또는 사용자 실제 사진} → 리뷰 제목 뒤, 리뷰 내용 전에 배치
- 보조1: {전후 비교 이미지 또는 착용/사용 사진} → 리뷰들 사이에 배치
- 보조2: {추가 사용자 사진 또는 디테일 사용 장면}

---

### 5. EMPATHY
**페인포인트**: {고민 체크리스트}
**공감 메시지**: {브랜드 공감 메시지}

**이미지 배치** (필수 1~2개) - 📍 위치: **텍스트 전**
- 메인: {공감 분위기 이미지 - 타겟 고객의 일상/고민 상황} → 이미지 먼저 → 페인포인트 체크리스트
- 보조1: {상황 연출 이미지 - 감정 표현}

---

### 6. DOUBT BREAKER (해당 시)
**고객 의심 1**: {의심}
**해소**: {해소 방법 + 근거}

**고객 의심 2**: {의심}
**해소**: {해소 방법 + 근거}

**고객 의심 3**: {의심}
**해소**: {해소 방법 + 근거}

**이미지 배치** (필수 2~3개) - 📍 위치: 텍스트 후 (증거 제시)
- 메인: {인증서, 테스트 결과, 특허증 등 공식 문서}
- 보조1: {비교 차트, 데이터 시각화}
- 보조2: {실험 장면, 제조 시설, 추가 증거}

---

### 7. TARGET AUDIENCE
**타겟 1**: {대상 + 추천 이유}
**타겟 2**: {대상 + 추천 이유}
**타겟 3**: {대상 + 추천 이유}

**이미지 배치** (필수 2~3개) - 📍 위치: **텍스트 전**
- 메인: {타겟 라이프스타일 대표 이미지} → 이미지 먼저 → "이런 분들께 추천" 텍스트
- 보조1: {다른 사용 장면 A}
- 보조2: {다른 타겟 그룹 사용 장면 B}

---

### 8. STORY
**스토리 유형**: {기원/브랜드/상징/장인}
**스토리 내용**: {150-250자 스토리}

**이미지 배치** (필수 2~3개) - 📍 위치: **텍스트 전** 또는 사이
- 메인: {스토리 핵심 이미지 - 원산지, 제작 현장 등} → 분위기 이미지 먼저
- 보조1: {역사적 사진 또는 배경 이미지}
- 보조2: {제작 과정 또는 원재료 이미지}

---

### 9. FOUNDER/TEAM (해당 시)
**인물**: {이름} · {직책}
**핵심 자격**: {학력/경력/수상 등}
**메시지**:
> {개인 스토리 또는 철학}

**이미지 배치** (필수 2~3개) - 📍 위치: **텍스트 전**
- 메인: {창업자/장인 프로필 사진 - 전문적이고 신뢰감 있는} → 얼굴 먼저 → 소개 텍스트
- 보조1: {작업 장면, 제작 현장에서의 모습}
- 보조2: {팀 사진, 공방/사무실 환경}

---

### 10. FEATURES
**특징 1**: {아이콘 + 제목 + 설명}
**특징 2**: {아이콘 + 제목 + 설명}
**특징 3**: {아이콘 + 제목 + 설명}

**이미지 배치** (필수 2~3개) - 📍 위치: **기능별 교차**
- 특징1 이미지: {기능 A 시각화} → 특징1 설명 →
- 특징2 이미지: {기능 B 시각화} → 특징2 설명 → ...
- (또는 대안) 📍 위치: 텍스트 후 - 모든 특징 설명 후 이미지 모아서 배치

---

### 11. MATERIALS
**소재 소개**: {왜 이 소재를 선택했는지}
**소재 속성 1**: {속성 + 설명}
**소재 속성 2**: {속성 + 설명}
**소재 속성 3**: {속성 + 설명}

**이미지 배치** (필수 2~3개) - 📍 위치: **텍스트 전**
- 메인: {소재 클로즈업 - 질감이 느껴지는}
- 보조1: {원재료 또는 원산지 이미지}
- 보조2: {소재 질감 디테일 또는 비교 이미지}

---

### 12. CRAFTSMANSHIP
**철학 인용구**: {브랜드 철학}
**제작 과정 1**: {단계 + 설명 + 하이라이트}
**제작 과정 2**: {단계 + 설명 + 하이라이트}
**제작 과정 3**: {단계 + 설명 + 하이라이트}

**이미지 배치** (필수 2~3개) - 📍 위치: **단계별 교차**
- 과정1 이미지: {1단계 제작 장면} → 과정1 설명 →
- 과정2 이미지: {2단계 제작 장면} → 과정2 설명 → ...
- (또는 대안) 📍 위치: 텍스트 전 - 공방 분위기 이미지 먼저 → 제작 과정 설명

---

### 13. FAQ
**Q1**: {질문} / **A1**: {답변}
**Q2**: {질문} / **A2**: {답변}
**Q3**: {질문} / **A3**: {답변}
**신뢰 뱃지**: {보증 항목들}

**이미지 배치**: 이미지 없음 (텍스트 중심 섹션) 또는
- 📍 위치: 텍스트 후 - 선택: {신뢰 뱃지/인증 마크 그래픽, 보증서 이미지}

---

### 14. OPTIONS
**안내 문구**: {선택 가이드}
**옵션 1**: {옵션명 + 특성 + 추천 대상}
**옵션 2**: {옵션명 + 특성 + 추천 대상}

**이미지 배치** (필수 2~3개) - 📍 위치: 텍스트 후
- 메인: {옵션별 비교 이미지 - 나란히 배치}
- 보조1: {옵션 A 상세 이미지}
- 보조2: {옵션 B 상세 이미지 또는 색상별 컷}

---

### 15. CTA
**프로모션**: {프로모션 내용}
**가격 표시**: {원가/할인가/할인율}
**CTA 버튼**: {버튼 문구}
**추가 안내**: {배송, 한정 수량 등}

**이미지 배치** (필수 1~2개) - 📍 위치: **텍스트 전**
- 메인: {제품 패키지 또는 전체 구성품 이미지} → 구성품 보여주고 → 가격/혜택 텍스트
- 보조1: {사은품 또는 특별 구성 이미지}

---

### 16. SPECS
**스펙 테이블**: {상세 규격 정보}
**주의사항**: {사용 전 확인 사항}

**이미지 구성**: 이미지 없음 (텍스트/표 중심 섹션)

---

### 17. FOOTER
**브랜드명 + 태그라인**

**이미지 구성**: 이미지 없음 (브랜드 로고만 사용)

---

(필요한 섹션만 선택하여 작성, 단계 분할 시 섹션 번호 조정)

## 이미지 배치 계획

### ⚠️ 이미지 수량 검증 (필수 확인)

```
📊 이미지 수량 체크
├── 총 섹션 수: {N}개
├── 필수 최소 이미지: {N}개 (섹션당 1개)
├── 권장 이미지 수: {N × 1.5 ~ N × 2.0}개
├── 현재 계획된 이미지: {실제 합계}개
└── 판정: ✅ 통과 / ⚠️ 미달 (보조 이미지 추가 필요)
```

### 섹션별 이미지 체크리스트

> 상태 표기: ✅ 보유 | 🎨 AI생성 필요 | ⬜ 불필요
> 위치 표기: 📍전 = 텍스트 전 | 📍후 = 텍스트 후 | 📍사이 = 텍스트 사이 | 📍교차 = 교차 배치

| 순서 | 섹션 | 📍위치 | 메인 이미지 | 보조 이미지 1 | 보조 이미지 2 | 소계 | 상태 |
|:----:|------|:------:|-------------|---------------|---------------|:----:|:----:|
| 1 | HOOK | - | - | - | - | 0 | ⬜ |
| 2 | HERO | 📍후 | {히어로샷 설명} | {다각도/연출} | - | 2 | 🎨 |
| 3 | SOCIAL PROOF | 📍후 | {인증뱃지} | {수상내역} | - | 1~2 | 🎨 |
| ... | ... | ... | ... | ... | ... | ... | ... |
| | **합계** | | **메인: {X}개** | **보조1: {Y}개** | **보조2: {Z}개** | **총 {합계}개** | |

### 이미지 합계 검증

```
📦 이미지 종류별 합계:
├── 메인 이미지: {X}개
├── 보조 이미지 1: {Y}개
├── 보조 이미지 2: {Z}개
└── 총합: {X+Y+Z}개

📐 수량 검증:
├── 섹션 수: {N}개
├── 최소 필요: {N}개 (섹션당 1개)
├── 권장 필요: {N × 1.5}개 ~ {N × 2.0}개
├── 현재 계획: {합계}개
└── 판정: ✅ 충족 / ⚠️ 미달 (보조 이미지 {부족분}개 추가 필요)
