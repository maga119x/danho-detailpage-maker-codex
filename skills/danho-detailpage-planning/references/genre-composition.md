# Genre-Based Section Composition (장르별 섹션 구성)

제품 장르에 따라 섹션 구성과 순서를 조정합니다. 모든 장르는 HOOK → HERO로 시작하고 FINAL-CLOSING → SPECS/FAQ → FOOTER로 마무리합니다. FINAL-CLOSING은 구매 버튼이나 행동 문구가 아니라 상품/결과 인상으로 닫는 섹션입니다.

## 섹션 수 가이드라인

| 구분 | 섹션 수 | 설명 |
|------|---------|------|
| **최소** | 12개 이상 | 설득력 있는 상세페이지의 최소 기준 |
| **권장** | 15~18개 | 충분한 깊이의 설득 구조 |
| **최대** | 25개 이하 | 너무 길면 이탈률 증가 |

> **중요**: "유연하게 구성"은 "최소로 구성"이 아닙니다. 각 설득 포인트를 충분히 펼치되, 불필요한 반복을 피하는 것이 유연한 구성입니다.

---

## 섹션 구성 매트릭스

| 장르 | 핵심 섹션 (⭐필수) | 강조 포인트 |
|------|-------------------|-------------|
| 뷰티/스킨케어 | MATERIALS, DOUBT BREAKER, REVIEWS | 성분 효과, 전후 비교 |
| 건강기능식품 | DOUBT BREAKER(앞배치), CRAFTSMANSHIP, MATERIALS | 안전성, 제조 과정 |
| 패션/의류 | MATERIALS, OPTIONS, REVIEWS | 소재, 사이즈 가이드 |
| 가전/전자기기 | FEATURES, SPECS, FAQ | 기능/스펙 중심 |
| 식품/음료 | STORY, MATERIALS, REVIEWS | 원산지, 생산자 |
| 인테리어/가구 | MATERIALS, CRAFTSMANSHIP, OPTIONS | 소재, 장인정신 |
| 선물/기념품 | STORY, EMPATHY, TARGET AUDIENCE | 의미/감성 |
| 크라우드펀딩 | FOUNDER/TEAM, DOUBT BREAKER, STORY | 창업자 신뢰, 펀딩 우려 해소 |
| 장인/핸드메이드 | FOUNDER/TEAM, CRAFTSMANSHIP, MATERIALS | 장인 경력, 제작 과정 |

## 장르별 권장 순서

> **참고**: 아래 섹션 수는 단계 분할을 적용한 권장 수입니다. 핵심 설득 포인트(⭐ 표시)는 2~3단계로 분할하여 깊이 있게 다루세요. 단계 분할에 대한 자세한 내용은 `persuasion-framework.md`를 참조하세요.

---

### 뷰티/스킨케어
**권장 섹션 수: 15~18개**

```
HOOK → HERO → SOCIAL PROOF
→ EMPATHY (2~3단계 분할: 페인포인트 → 공감 → 희망)
→ ⭐DOUBT BREAKER (2~3단계 분할: 의심 인정 → 근거 → 보장)
→ ⭐MATERIALS (2~3단계 분할: 왜 중요 → 성분 설명 → 효과)
→ FEATURES
→ ⭐REVIEWS → TARGET → FAQ → OPTIONS
→ FINAL-CLOSING → SPECS → FOOTER
```

---

### 건강기능식품
**권장 섹션 수: 16~20개**

```
HOOK → HERO
→ ⭐DOUBT BREAKER (3단계 분할: 안전성 우려 인정 → 인증/테스트 → 환불 보장) ← 앞배치 중요
→ SOCIAL PROOF
→ EMPATHY (2단계 분할)
→ ⭐MATERIALS (3단계 분할: 왜 이 성분 → 성분 설명 → 임상 결과)
→ ⭐CRAFTSMANSHIP (2단계 분할: 제조 철학 → 제조 과정)
→ FEATURES → REVIEWS → TARGET → FAQ → OPTIONS
→ FINAL-CLOSING → SPECS → FOOTER
```

---

### 패션/의류
**권장 섹션 수: 12~15개**

```
HOOK → HERO → SOCIAL PROOF
→ ⭐MATERIALS (2~3단계 분할: 왜 소재가 중요 → 소재 특성 → 착용감)
→ FEATURES
→ ⭐OPTIONS(사이즈 가이드 상세히)
→ ⭐REVIEWS(착용샷 다양하게)
→ TARGET → FAQ
→ FINAL-CLOSING → SPECS → FOOTER
```
- STORY, CRAFTSMANSHIP 선택적 추가 가능

---

### 가전/전자기기
**권장 섹션 수: 12~16개**

```
HOOK → HERO → SOCIAL PROOF
→ ⭐FEATURES (2~3단계 분할: 문제 인식 → 기능 설명 → 사용 효과) ← 핵심!
→ ⭐SPECS (상세하게)
→ REVIEWS
→ ⭐FAQ(기술 Q&A 충실히)
→ OPTIONS
→ FINAL-CLOSING → FOOTER
```
- STORY, EMPATHY는 선택적 (프리미엄 제품일 경우 추가)

---

### 식품/음료
**권장 섹션 수: 14~17개**

```
HOOK → HERO → SOCIAL PROOF
→ ⭐STORY (2단계 분할: 원산지 배경 → 생산자 이야기)
→ ⭐MATERIALS (2~3단계 분할: 왜 원재료가 중요 → 원재료 소개 → 맛의 차이)
→ FEATURES
→ ⭐REVIEWS(맛 후기 다양하게)
→ TARGET → OPTIONS → FAQ
→ FINAL-CLOSING → SPECS → FOOTER
```

---

### 인테리어/가구
**권장 섹션 수: 15~18개**

```
HOOK → HERO → SOCIAL PROOF → STORY
→ ⭐MATERIALS (2~3단계 분할: 소재의 중요성 → 소재 특성 → 내구성)
→ ⭐CRAFTSMANSHIP (2단계 분할: 제작 철학 → 제작 과정)
→ FEATURES
→ ⭐OPTIONS(사이즈/색상 상세 가이드)
→ REVIEWS → FAQ(설치/배송 상세히)
→ FINAL-CLOSING → SPECS → FOOTER
```

---

### 선물/기념품
**권장 섹션 수: 14~17개**

```
HOOK → HERO
→ ⭐STORY (2~3단계 분할: 선물의 의미 → 제품 스토리 → 받는 사람의 감동)
→ ⭐EMPATHY (2단계 분할: 선물 고민 공감 → 완벽한 선물 제안)
→ ⭐TARGET AUDIENCE (선물 대상별 추천 상세히)
→ SOCIAL PROOF → FEATURES → MATERIALS
→ OPTIONS(포장 옵션 상세히)
→ REVIEWS → FAQ
→ FINAL-CLOSING → SPECS → FOOTER
```

---

### 크라우드펀딩/스타트업
**권장 섹션 수: 16~20개**

```
HOOK → HERO
→ ⭐FOUNDER/TEAM (2단계 분할: 창업자 소개 → 팀 역량)
→ ⭐STORY (2단계 분할: 왜 만들게 되었나 → 개발 과정)
→ EMPATHY
→ ⭐DOUBT BREAKER (3단계 분할: 펀딩 우려 인정 → 진행 현황/증거 → 보장 정책)
→ FEATURES → MATERIALS/CRAFTSMANSHIP
→ OPTIONS(리워드 상세 설명)
→ TARGET → FAQ(배송/환불 상세히)
→ FINAL-CLOSING → SPECS → FOOTER
```

**펀딩 특화 DOUBT BREAKER 포인트**:
- 배송 지연 우려 → 현재 진행 단계 공개
- 완성도 우려 → 완제품 촬영, 환불 보장
- A/S 우려 → 무상 기간, 고객센터 운영

---

### 장인/핸드메이드
**권장 섹션 수: 14~17개**

```
HOOK → HERO
→ ⭐FOUNDER/TEAM (2~3단계 분할: 장인 소개 → 경력/철학 → 작업 모습)
→ STORY
→ ⭐CRAFTSMANSHIP (2~3단계 분할: 장인정신 철학 → 제작 과정 → 디테일)
→ ⭐MATERIALS (2단계 분할: 소재 선택 기준 → 소재 특성)
→ FEATURES → REVIEWS → OPTIONS → FAQ
→ FINAL-CLOSING → SPECS → FOOTER
```

**특화 포인트**: 한정 수량/제작 기간 명시
