## 이미지 수량 필수 기준

```
┌─────────────────────────────────────────────────────────────┐
│  🚨 기획서 작성 시 반드시 확인                               │
├─────────────────────────────────────────────────────────────┤
│  • 기획서의 이미지 계획이 6~8개에 그치면 안 됩니다!          │
│  • 14개 섹션 기준 최소 14개, 권장 21~28개 이미지 필요        │
│  • 각 섹션마다 메인 + 보조 1~2개를 명시적으로 작성하세요     │
└─────────────────────────────────────────────────────────────┘
```

### 수량 기준

| 구분 | 이미지 수 | 계산 방법 | 판정 |
|------|-----------|-----------|------|
| **최소** | 섹션 수 × 1.0 | 14개 섹션 → 14개 이미지 | ⚠️ 미달 시 수정 필요 |
| **권장** | 섹션 수 × 1.5~2.0 | 14개 섹션 → 21~28개 이미지 | ✅ 목표 수준 |
| **풍부함** | 섹션 수 × 2.0~2.5 | 14개 섹션 → 28~35개 이미지 | ⭐ 프리미엄 수준 |

> **⚠️ 중요**: 이미지가 적으면 텍스트 위주의 밋밋한 페이지가 됩니다.
> 이커머스 상세페이지는 **시각적 설득**이 핵심이므로 **최소 권장 수량(섹션 수 × 1.5) 이상**을 사용하세요.
>
> **❌ 흔한 실수**: 메인 이미지만 계획하고 보조 이미지를 생략함 → 이미지 부족
> **✅ 올바른 방법**: 각 섹션마다 "메인 + 보조1 + (보조2)" 형식으로 구체적으로 명시

## 섹션별 이미지 배치 권장

| 섹션 | 메인 이미지 | 보조 이미지 1 | 보조 이미지 2 | 권장 합계 | 권장 위치 |
|------|-------------|---------------|---------------|-----------|-----------|
| HOOK | 배경 분위기 | - | - | 0~1 | 텍스트 배경 |
| HERO | 히어로샷 | 제품 다각도 | - | 1~2 | 텍스트 후 |
| SOCIAL PROOF | 인증 뱃지 | 수상 내역 | - | 0~2 | 텍스트 후 |
| REVIEWS | 리뷰 캡처 | 전후 비교 | 사용자 사진 | 2~3 | **리뷰 사이** |
| EMPATHY | 공감 분위기 | 상황 연출 | - | 1~2 | **텍스트 전** |
| DOUBT BREAKER | 인증서/테스트 | 비교 차트 | 추가 증거 | 2~3 | 텍스트 후 |
| TARGET | 라이프스타일 | 사용 장면 A | 사용 장면 B | 2~3 | **텍스트 전** |
| STORY | 스토리 이미지 | 원산지/배경 | 제작 현장 | 2~3 | **텍스트 전** 또는 사이 |
| FOUNDER/TEAM | 프로필 사진 | 작업 장면 | 팀 사진 | 2~3 | **텍스트 전** |
| FEATURES | 기능 시각화 | 디테일컷 | 사용 장면 | 2~3 | **기능별 교차** |
| MATERIALS | 소재 클로즈업 | 원재료 | 질감 디테일 | 2~3 | **텍스트 전** |
| CRAFTSMANSHIP | 제작 과정 | 공방/환경 | 디테일 장면 | 2~3 | **단계별 교차** |
| FAQ | 신뢰 뱃지 | 인증 마크 | - | 0~2 | 텍스트 후 |
| OPTIONS | 옵션 비교 | 옵션 상세 A | 옵션 상세 B | 2~3 | 텍스트 후 |
| FINAL CLOSING | 제품 결과 장면 | 패키지/사용 후 생활 장면 | - | 1~2+ | **FULL_IMAGE** |
| SPECS | - | - | - | 0 | - |
| FOOTER | - | - | - | 0 | - |

## 이미지 배치 위치 가이드

**이미지는 섹션 끝에만 있을 필요가 없습니다.** 메시지 전달 효과에 따라 유연하게 배치합니다.

### 위치 유형

| 위치 | 설명 | 효과 | 권장 섹션 |
|------|------|------|-----------|
| **텍스트 전** | 이미지 → 텍스트 | 시각적 임팩트 먼저, 설명은 후 | EMPATHY, TARGET, STORY, MATERIALS, FOUNDER |
| **텍스트 후** | 텍스트 → 이미지 | 설명 후 증거 제시 | HERO, FAQ, OPTIONS, DOUBT BREAKER |
| **텍스트 사이** | 텍스트 → 이미지 → 텍스트 | 이미지로 단락 구분 | REVIEWS, STORY |
| **교차 배치** | 이미지 → 텍스트 → 이미지 → ... | 스크롤 리듬감 | FEATURES, CRAFTSMANSHIP |

### 배치별 설득 효과

```
┌─────────────────────────────────────────────────────────────┐
│  📷 텍스트 전 (Image-First)                                  │
│  "보여주고 → 설명하기"                                        │
│  • 시각적 임팩트가 강한 제품                                  │
│  • 분위기/감성 전달이 중요한 경우                             │
│  • 타겟이 공감할 상황 이미지                                  │
├─────────────────────────────────────────────────────────────┤
│  📝 텍스트 후 (Text-First)                                   │
│  "설명하고 → 증거 보여주기"                                   │
│  • 기술적 설명이 필요한 기능                                  │
│  • 스펙/인증 정보 후 이미지 증거                              │
│  • 가격/혜택 설명 후 패키지 이미지                            │
├─────────────────────────────────────────────────────────────┤
│  🔀 교차 배치 (Interspersed)                                 │
│  "이미지 → 설명 → 이미지 → 설명"                              │
│  • 여러 기능/특징을 하나씩 설명                               │
│  • 제작 과정 단계별 설명                                      │
│  • 스크롤 지루함 방지                                         │
├─────────────────────────────────────────────────────────────┤
│  📊 텍스트 사이 (Between)                                    │
│  "도입 → 이미지 → 상세 설명"                                  │
│  • 제목/리드 → 이미지 → 본문                                  │
│  • 리뷰 제목 → 리뷰 이미지 → 리뷰 내용                        │
└─────────────────────────────────────────────────────────────┘
```

### 섹션별 권장 배치 패턴

**EMPATHY (공감) - 텍스트 전 권장**
```
[이미지: 고민하는 사람/상황 분위기]
↓
"혹시 이런 고민 있으셨나요?"
페인포인트 체크리스트...
```

**TARGET (타겟) - 텍스트 전 권장**
```
[이미지: 타겟 라이프스타일]
↓
"이런 분들께 추천드립니다"
타겟 그룹별 설명...
```

**FEATURES (특징) - 교차 배치 권장**
```
[이미지: 기능 A 시각화]
↓
"🔇 액티브 노이즈 캔슬링"
설명...
↓
[이미지: 기능 B 시각화]
↓
"🔋 30시간 배터리"
설명...
```

**REVIEWS (리뷰) - 텍스트 사이 권장**
```
섹션 제목: "실제 사용 후기"
↓
[이미지: 리뷰 캡처/사용자 사진]
↓
리뷰 내용들...
↓
[이미지: 추가 사용 장면]
```

**FINAL CLOSING (마감 인상) - FULL_IMAGE 필수**
```
[이미지: 제품 결과 장면/사용 후 생활 장면]
↓
제품명 재확인, 구매 행동 없이 조용한 결과 마감 카피...
```

## 이미지 다양성 원칙

하나의 섹션에서 다양한 시각으로 설득력을 높이세요:

| 이미지 유형 | 역할 | 예시 |
|-------------|------|------|
| **메인 이미지** | 섹션의 핵심 메시지 전달 | 제품 전체샷, 기능 시각화 |
| **디테일컷** | 품질/마감 강조 | 봉제선, 소재 질감, 조작부 |
| **사용 장면** | 실생활 연결 | 착용샷, 사용 중인 모습 |
| **비교 이미지** | 차별점 강조 | 전후 비교, 경쟁사 비교 |
| **인증/증거** | 신뢰 구축 | 인증서, 테스트 결과 |
| **다각도** | 제품 이해 | 정면, 측면, 후면, 45도 |

## 보조 이미지 활용 가이드

보조 이미지는 다음 용도로 활용합니다:
- **디테일컷**: 제품의 세부 특징 강조
- **사용 장면**: 실제 사용 상황 연출
- **전후 비교**: 효과 입증 (특히 뷰티/건강 제품)
- **원재료/원산지**: 품질 신뢰 구축
- **인증서/테스트 결과**: 의심 해소
- **다각도 촬영**: 제품 전체 이해

> **Tip**: 섹션당 2~3개 이미지를 적극 활용하세요. 하나의 메인 이미지만으로는 시각적 설득력이 부족합니다.

## 이미지 유형 설명

| 유형 | 설명 | Codex 네이티브 이미지 생성 |
|------|------|-------------------|
| **제품 사진** | 실제 촬영한 듯한 제품 이미지 | `danho-imageprompt-helper` → 사진 유형 |
| **히어로 영역 디자인** | 제품 + 레이아웃 + 텍스트 조합 웹 디자인 | `danho-imageprompt-helper` → 디자인 유형 |
| **미니멀 웹포스터** | 감성적 비주얼과 타이포그래피 중심의 세련된 포스터 | `danho-imageprompt-helper` → 디자인 유형 |
| **이미지 오버레이 타이포그래피** | 풀블리드 배경 이미지 위에 한글 텍스트 오버레이 | `danho-imageprompt-helper` → 디자인 유형 (오버레이형) |
| **디테일컷** | 제품의 세부 특징 강조 클로즈업 | `danho-imageprompt-helper` → 사진 유형 |
| **라이프스타일** | 제품을 사용하는 일상 장면 | `danho-imageprompt-helper` → 사진 유형 |
| **전후 비교** | Before/After 효과 입증 이미지 | `danho-imageprompt-helper` → 사진 유형 |

> 🎨 **AI 이미지 생성**: 이미지가 필요한 경우 `danho-imageprompt-helper` 스킬을 호출하여 Codex 네이티브 이미지 생성용 프롬프트를 만들고 이미지를 생성할 수 있습니다.

## 이미지 수량 검증 템플릿

기획서 작성 시 아래 템플릿을 사용하여 이미지 수량을 검증하세요:

```
📊 이미지 수량 체크
├── 총 섹션 수: {N}개
├── 필수 최소 이미지: {N}개 (섹션당 1개)
├── 권장 이미지 수: {N × 1.5 ~ N × 2.0}개
├── 현재 계획된 이미지: {실제 합계}개
└── 판정: ✅ 통과 / ⚠️ 미달 (보조 이미지 추가 필요)
```

### 부족 이미지 보완 우선순위

| 우선순위 | 섹션 | 추가 권장 이미지 |
|:--------:|------|------------------|
| 1순위 | FEATURES | 기능별 상세 이미지, 사용 중인 모습 |
| 2순위 | MATERIALS | 소재 질감 클로즈업, 원재료 원산지 |
| 3순위 | REVIEWS | 추가 사용자 사진, 다양한 사용 장면 |
| 4순위 | TARGET | 다른 타겟 그룹의 사용 장면 |
| 5순위 | STORY | 제작 과정 단계별 이미지 |
