# Korean Copywriting Templates

## Hook Section Templates

### Question Hook (질문형)
```
왜 어떤 사람은 늘 {positive_outcome}일까요?
{product}의 비밀, 이제 공개합니다.
```

### Bold Statement Hook (선언형)
```
{time_period}간 전해 내려온 {benefit}의 비밀
당신의 {target_area}을 바꿔줄 단 하나의 {product_type}
```

### Pain Point Hook (고통형)
```
{pain_point}로 고민하셨나요?
이제 그 걱정, 내려놓으세요.
```

## Review Templates

### Standard Review
```

```

## FAQ Templates

### Concern-Reassurance Format
```
Q: {concern_question}
A: {direct_answer}. {proof_point}. {additional_reassurance}.
```

### Common Questions
- 품질/진품 여부: "정품/천연 {material} 맞나요?"
- 사이즈/핏: "{size} 맞을까요?"
- 관리/내구성: "쉽게 {problem} 되지 않나요?"
- 선물 포장: "선물용 포장 되나요?"
- 교환/반품: "마음에 안 들면 반품 가능한가요?"

## Doubt Breaker Templates

### Standard Format
```
💭 "{고객의 의심/우려}"
{해소 방법 + 객관적 근거}
[강조할 수치/뱃지]
```

### Comparison Format
```
💭 "{가격/가치 의심}"
{비교 대상}과 비교하면:
- {비교 포인트 1}
- {비교 포인트 2}
결론: {가성비/가치 강조}
```

### Guarantee Format
```
💭 "{적합성/품질 의심}"
{안전성/테스트 결과}
그래도 걱정되시면? → {보장 정책}
```

### Crowdfunding Specific
```
💭 "펀딩이라 배송이 늦어지지 않을까?"
현재 {진행 단계} 완료.
펀딩 종료 후 {기간} 내 순차 배송.
[진행 상황 투명 공개]

💭 "스타트업이라 A/S가 걱정돼요"
{기간} 무상 A/S + 평생 유상 수리.
고객센터 {운영 정보}
```

## Genre-Specific Phrases

### 뷰티/스킨케어
- "피부가 달라지는 경험"
- "과학이 증명한 효과"
- "민감성 피부도 안심"
- "트러블 없이 순하게"

### 건강기능식품
- "몸이 먼저 알아봅니다"
- "자연에서 찾은 답"
- "매일 한 알로 시작하는 변화"
- "과학적 배합, 자연의 원료"

### 패션/의류
- "입는 순간 달라지는 실루엣"
- "편안함과 스타일, 둘 다"
- "세탁해도 변하지 않는 핏"
- "매일 입고 싶은 옷"

### 가전/테크
- "기술이 일상을 바꿉니다"
- "복잡한 건 우리가, 편리함은 당신이"
- "한 번 쓰면 못 돌아갑니다"
- "스마트한 선택"

### 크라우드펀딩
- "함께 만들어가는 제품"
- "첫 번째 서포터가 되어주세요"
- "당신의 지지가 혁신이 됩니다"
- "아이디어에서 현실로"

### 장인/핸드메이드
- "손끝에서 전해지는 온기"
- "시간이 만들어낸 가치"
- "기계로는 불가능한 것들"
- "느리지만 오래가는"

## 이미지 제안 템플릿

### 히어로 이미지 유형별 제안

#### 제품 사진 (Photo)
```
📷 제품 사진 제안
- 유형: 제품 단독샷 / 연출샷 / 라이프스타일
- 배경: {배경 설명}
- 앵글: {촬영 각도}
- 조명: {조명 유형}
- 분위기: {무드 키워드}

🎨 Codex 네이티브 이미지 생성 필요 시: `danho-imageprompt-helper` 스킬 호출
```

#### 히어로 영역 디자인 (Hero Design)
```
🖼️ 히어로 영역 디자인 제안
- 유형: e-commerce hero design
- 레이아웃: {분할 구조 - 예: 좌측 제품, 우측 텍스트}
- 텍스트 영역:
  - 헤드라인: "{한글 헤드라인}"
  - 서브카피: "{한글 서브카피}"
- 컬러 스킴: {색상 조합}
- 무드: {브랜드 톤앤매너}

🎨 Codex 네이티브 이미지 생성 필요 시: `danho-imageprompt-helper` 스킬 호출 (디자인 유형)
```

#### 미니멀 웹포스터 (Minimal Web Poster)
```
🎨 미니멀 웹포스터 제안
- 유형: minimalist web poster design
- 비주얼: {핵심 비주얼 요소}
- 타이포그래피:
  - 메인 카피: "{한글 메인 카피}"
  - 폰트 스타일: {고딕/명조/임팩트}
- 컬러: {메인 컬러 + 포인트 컬러}
- 여백: {화이트 스페이스 활용 방식}

🎨 Codex 네이티브 이미지 생성 필요 시: `danho-imageprompt-helper` 스킬 호출 (디자인 유형)
```

#### 이미지 오버레이 타이포그래피 (Image Overlay Typography)
```
✨ 이미지 오버레이 타이포그래피 제안
- 유형: full-bleed background with text overlay
- 배경 이미지: {배경 사진/그래픽 설명}
- 오버레이 처리: {그라데이션/블러/다크닝}
- 텍스트:
  - 헤드라인: "{강렬한 한글 메시지}"
  - 위치: {center/top/bottom}
  - 크기: {extra-large/large}
  - 폰트: {bold sans-serif/elegant serif}
- 대비: {텍스트 가독성을 위한 처리}

🎨 Codex 네이티브 이미지 생성 필요 시: `danho-imageprompt-helper` 스킬 호출 (디자인 유형 - 오버레이형)
```

### 이미지 유형 선택 가이드

| 제품 특성 | 권장 이미지 유형 |
|----------|-----------------|
| 제품 자체가 시각적으로 매력적 | 제품 사진 |
| 브랜드 메시지/프로모션 강조 | 히어로 영역 디자인 |
| 감성적 브랜드 무드 전달 | 미니멀 웹포스터 |
| 강렬한 첫인상/메시지 중심 | 이미지 오버레이 타이포그래피 |
| 고급스러운 느낌 | 미니멀 웹포스터 / 오버레이 타이포그래피 |
| 정보 전달 중심 | 히어로 영역 디자인 |
