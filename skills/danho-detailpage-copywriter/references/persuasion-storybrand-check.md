# Persuasion And StoryBrand Check

Use these as high-level copywriting principles. Do not quote or imitate copyrighted text.

## Ethical Cialdini-Inspired Checks

- Reciprocity: Give the buyer a useful checklist, install guide, or compatibility warning before asking for purchase.
- Commitment and consistency: Let the buyer make a small fit decision early. Example: `우리 집 문에 자석이 붙나요?`
- Social proof: Use real reviews, real usage numbers, or verified buyer feedback when supplied. A review/testimonial section is still mandatory; if real reviews are missing, use neutral replacement-ready dummy review cards without fabricated specifics and keep the replacement marker internal.
- Authority: Use certifications, tests, awards, or expert claims only when supplied by the product facts.
- Liking and unity: Mirror the buyer's actual situation in plain Korean. Do not flatter or over-identify.
- Scarcity: Use deadlines, stock limits, or discounts only when factual.

## StoryBrand-Style Buyer Journey

- Customer is hero: copy centers on the buyer's problem and desired outcome.
- Brand/product is guide: the product helps the buyer take a simple action; it is not the hero of the story.
- Opening bridge: the first section introduces the desired result, and the second section makes the buyer recognize the lived problem or question behind that result.
- Clear plan: show the 2-3 step path to use or buy.
- Call to action: tell the buyer the next action clearly.
- Success state: show what daily life feels like after purchase.
- Stakes: mention the avoidable inconvenience without fearmongering.

## Customer-As-Subject Rule

For each section, identify the sentence center:

| If the center is... | Then rewrite toward... |
|---|---|
| brand achievement | buyer relief or easier action |
| product feature | buyer use moment |
| manufacturing effort | buyer confidence |
| abstract quality | concrete daily outcome |
| seller promise | buyer's next step |

Good detail-page copy should make the buyer think:

- "내 상황에 맞겠다"
- "이걸 쓰면 이 불편이 줄겠다"
- "구매 전에 뭘 확인해야 하는지 알겠다"
- "사용 방법이 어렵지 않겠다"

If the sentence mainly says "we made / we provide / this product has", rewrite it.

## Opening Bridge Check

Before polishing section-level Korean, check the first two sections together:

1. Does section 01 create a clear product/result promise?
2. Does section 02 continue the same buyer moment instead of starting a new topic?
3. Can the buyer feel "that is my situation" within section 02?
4. Is there enough scene, emotion, or visual substance for empathy?

If any answer is no, revise the opening pair before scoring the rest of the page.

## Benefit Audit Questions

For each visible line, ask:

1. Does this sentence describe the buyer's situation or only the product?
2. Is the buyer or buyer outcome the semantic subject?
3. Is the benefit concrete enough to picture?
4. Is every proof claim backed by supplied facts?
5. Would a Korean customer-facing seller naturally say this sentence?
6. Does this sentence move the buyer to the next decision?

If any answer is no, rewrite before coding.
