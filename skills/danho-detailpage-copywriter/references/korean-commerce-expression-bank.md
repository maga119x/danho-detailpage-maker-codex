# Korean Commerce Expression Bank

Use this when rewriting visible Korean copy for ecommerce detail pages. Prefer practical, spoken buyer-facing Korean over report-like wording.

## Tone Defaults

- Hooks and benefit copy: friendly polite `해요체`.
- Specs, limits, compatibility, delivery, warranty, and FAQ facts: clear 안내형 `합니다체`.
- CTA: direct but not pushy.
- Avoid cute language unless the brand already uses it.

## Reliable Patterns

| Intent | Pattern |
|---|---|
| Fit check | `구매 전, 이 부분만 꼭 확인해 주세요!` |
| Compatibility | `{조건}라면 {제품}을 사용할 수 있어요.` |
| Simple action | `{행동 1}, {행동 2}, 바로 {결과}.` |
| Daily benefit | `{상황} 때 {불편}이 줄어들어요.` |
| Anxiety reduction | `{걱정}이 부담이라면 {확인/방법}부터 보세요.` |
| Limit | `{불가 조건}에는 사용할 수 없습니다.` |
| Option choice | `{사용 상황}이 많다면 {옵션}을 선택해 주세요.` |
| Final CTA | `{조건}에 맞는다면 지금 옵션을 선택해 주세요.` |

## Feature To Benefit Conversions

| Feature-first | Buyer-first rewrite |
|---|---|
| 강력한 자석 고정력 | 구멍 내지 않고도 필요한 순간에 문을 잡아둘 수 있어요. |
| 무타공 설치 | 전세집 현관문에도 자국 걱정을 줄이고 붙여 쓸 수 있어요. |
| 높이 조절 가능 | 바닥 높이가 달라도 문에 맞춰 사용할 수 있어요. |
| 스테인리스 소재 | 매일 발로 내려도 쉽게 휘청이지 않도록 단단한 소재를 확인해 주세요. |
| 간편 설치 | 붙이고, 내리고, 바로 고정하세요. |

## Report-Like To Commerce-Like

| Avoid | Use |
|---|---|
| 구매 전에는 이 조건만 먼저 확인하세요 | 구매 전, 이 부분만 꼭 확인해 주세요! |
| 설치 방식은 간단합니다 | 자석이 붙는 문이라면 간단하게 설치할 수 있어요. |
| 생활 장면에서 설득됩니다 | 짐 들고 들어올 때, 문이 다시 닫히지 않게. |
| 제품의 편의성을 제공합니다 | 손이 부족할 때도 발로 간단히 고정할 수 있어요. |
| 사용자 경험을 향상시킵니다 | 자주 불편했던 순간이 조금 더 편해져요. |
| 고객의 니즈를 충족합니다 | 이런 상황이라면 특히 잘 맞아요. |

## Source-Brief To Buyer Copy

Use this when source planning text sounds strategic but awkward as visible copy.

| Avoid as visible copy | Why it fails | Rewrite direction |
|---|---|---|
| 요리 좀 해보고 싶은 남자에게, 첫 칼은 중식도면 충분합니다 | stiff and over-declarative | 고기 썰고, 파 다지고, 마늘 으깨는 일까지 한 자루로. |
| 의외로 칼부터 답답합니다 | vague and unnatural | 고기 썰 때 칼이 밀리면 요리하기 싫어지죠. |
| "요리 장비"처럼 느껴지지 않았던 걸지도 모릅니다 | quoted concept, memo-like | 손에 맞는 칼 하나면, 요리가 한결 편해집니다. |
| 일반 식도는 익숙합니다 | flat setup sentence | 늘 쓰던 식칼로도 되지만, 많이 썰고 옮길 때는 한계가 느껴져요. |
| 칼 하나가 아닙니다 | slogan fragment feels forced | 칼과 관리용품이 함께 들어 있는 입문 세트예요. |
| 첫 중식도를 위한 스타터 세트입니다 | acceptable as product definition, weak as headline | 첫 중식도라면 관리용품까지 챙겨보세요. |
| 장비감 | internal strategy label | 손에 잡히는 묵직한 사용감 |
| 세트 완성도 | internal strategy label | 처음에 필요한 게 함께 들어 있음 |
| 가격 합리화 | internal planning label | 숫자 가격을 쓰지 말고 구성 가치와 옵션 영역 확인으로 설명 |

## Expression Polish Repairs

| Avoid | Why it fails | Use |
|---|---|---|
| 고기 썰고, 파 다지고, 마늘 으깨는 요리부터 | Incomplete headline ending. | 고기 썰고, 파 다지고, 마늘 으깨는 일까지 한 자루로 |
| 손에 맞는 도구가 먼저 체감됩니다 | Wrong subject for `체감`. | 손에 맞는 도구를 쓰면 그 차이가 바로 느껴집니다 |
| 손에 잡히는 안정감이 체감됩니다 | Translationese and repeated `체감`. | 손에 잡히는 안정감이 확실히 다릅니다 |
| 묵직한 실루엣 | `실루엣` is visual shape, not weight or handling. | 묵직한 무게감 |
| 리듬 있게 손질 | Decorative translationese. | 쭉쭉 썰기 / 빠르게 썰기 |
| 도마 앞이 달라집니다 | Slightly translated and abstract. | 요리가 한결 편해집니다 |
| 시작 구성이 완성됨 | Stiff nominal ending. | 처음에 필요한 게 다 있어요 |
| 첫 중식도 관리 기준 확인 | Bureaucratic. | 처음 쓸 때 알아둘 관리법 |
| 절단 가능 여부 | Manual-style phrasing. | 자를 수 있는지는 |

## Spec/Internal Terms To Rewrite

| Avoid in visible copy | Use |
|---|---|
| 본품 | 칼 / 중식도 |
| 관리 구성 | 관리용품 / 관리에 필요한 구성 |
| 보관 구성 | 보관용품 / 보호 커버 |
| 최종 상품 스펙 | 상품 정보 |
| 강재 | 칼날 소재 / 재질 |
| 제공 이미지 | remove from visible copy |
| 추가 확인 필요 | move to proof log |

## Headline Shapes

- Situation: `{상황} 때, {불편}이 반복된다면`
- Benefit: `{행동}만으로 {좋아지는 결과}`
- Check: `우리 집 {조건}부터 확인해 주세요`
- Relief: `{걱정}은 줄이고, {행동}은 더 쉽게`
- Choice: `{사용 상황}에 맞춰 옵션을 선택하세요`

## CTA Shapes

- `우리 집 문에 맞는지 확인하고 옵션을 선택해 주세요.`
- `현재 혜택은 옵션 영역에서 확인해 주세요.`
- `{조건}에 맞는다면 오늘부터 간단하게 사용해 보세요.`

## Avoid

- Unsupported hype: `검증된`, `입소문 난`, `수많은 분들이 선택한`.
- Empty superlatives: `완벽한`, `혁신적인`, `프리미엄 경험`.
- Mystery hooks: `비밀 공개`, `아무도 몰랐던`, `단 하나의`.
- Seller achievement: `저희가 개발했습니다`, `브랜드가 제안합니다`, `담았습니다`.
- Noun-heavy phrases: `설치의 진행`, `사용성의 향상`, `경험의 제공`.
- Internal labels as visible copy: `구매 판단 요약`, `불편의 순간`, `설치 방식`, `소재감`.
- Source-brief phrases as visible copy: `장비감`, `세트 완성도`, `가격 합리화`, `~면 충분합니다`, `의외로 ~ 답답합니다`, `"~처럼" 느껴지지 않았던`.
- Wrong collocations: `도구가 체감됩니다`, `안정감이 체감됩니다`, `쓰임이 느껴집니다`.
- Production notes as visible copy: image filenames, `제공 이미지`, `추가 확인 필요`, `상세페이지 공개 전`, `REPLACE_CANDIDATE`, `SUPPORT_CANDIDATE`.
- Sales channel copy: `스마트스토어`, `쿠팡`, `자사몰`, `판매 채널`, `채널별 구성`, repeated `구매 페이지에서 확인`.

## Review And Proof

- A review/testimonial section is mandatory in a real detail page.
- Use real reviews when supplied.
- If no real review exists, write neutral replacement-ready dummy review cards and mark them internally as `REVIEW_PLACEHOLDER_REPLACE_REQUIRED`.
- Use check-oriented review headlines: `실제 사용자 후기를 확인해 보세요` for supplied real reviews, or `사용 후기를 확인해 보세요` / `사용 후기로 확인할 포인트` for replacement-ready cards.
- Avoid staged review headlines such as `먼저 써본 사람이 말해요`, `먼저 써본 사람들의 이야기`, and `써본 사람은 이렇게 말합니다`.
- Do not expose placeholder warnings such as `더미 리뷰`, `실제 리뷰 없음`, `교체 예정`, or `NEEDS_PROOF` in visible copy.
- Do not invent age, names, locations, dates, review stars, review counts, purchase counts, awards, numbers, certifications, deadlines, or stock limits.
