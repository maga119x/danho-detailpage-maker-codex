# Korean-First Expression Gate

Use this when Korean copy is understandable but still sounds translated, stiff, strategic, or not like something a Korean seller would actually say.

## Core Principle

Strategy can use planning concepts. Visible copy cannot.

Before writing display copy, convert every strategy into Korean customer/seller speech. The output should sound like a real buyer thought or a seller explanation, not a translated marketing framework.

## Layer Separation

| Layer | Allowed | Forbidden |
|---|---|---|
| Strategy notes | target desire, before/after, value stack, mechanism, friction, conversion force, visual proof | n/a |
| Customer verbatim strategy | `칼만 사면 또 뭘 사야 할지 찾아봐야 하잖아`, `이 정도면 처음 시작하기 편하겠다` | abstract nouns without a speaker |
| Visible copy | short Korean sentences a seller could say aloud | strategy terms, translated abstractions, internal labels |

If a strategy term appears in visible copy, revise.

## Strategy-Term Leak List

These are planning terms. Do not use them in visible buyer-facing copy unless the phrase is genuinely everyday Korean in that exact context.

- `장비감`
- `전환`
- `before/after`
- `애프터 상태`
- `셀프 이미지`, `자기 인식`
- `메커니즘`
- `가치 프레임`
- `value stack`
- `conversion force`
- `friction`, `마찰`
- `동선`
- `흐름을 줄이다`
- `선택을 줄이다`
- `구매 저항`
- `가격 방어`
- `핵심 셀링포인트`
- `차별점`

Better visible-copy directions:

| Strategy wording | Customer/seller speech |
|---|---|
| 선택을 줄이는 세트 | 처음부터 같이 있으면 따로 찾을 일이 줄어요 |
| 도마 위 동선을 줄입니다 | 썬 재료를 바로 팬으로 옮길 수 있어요 |
| self-image 강화 | 좋은 도구 하나 갖췄다는 기분이 들어요 |
| value stack | 칼만이 아니라 같이 필요한 것까지 들어 있어요 |
| 메커니즘 증명 | 어떻게 편해지는지 장면으로 보여주세요 |

## Korean Sentence Skeleton Rules

Visible copy should follow Korean speech skeletons.

- Prefer people, situations, or actions as the subject. Avoid abstract objects as the subject of selling copy.
- Prefer verbs over nominal endings. `관리 가능` -> `관리할 수 있어요`.
- Keep one sentence to 1-2 clauses. Split longer sentences.
- Avoid 3 or more modifiers before one noun.
- Avoid abstract noun chains: `첫 구매의 선택을 줄였습니다`, `재료 준비의 흐름을 줄여주는 도구`.
- Avoid object-subject structures copied from English: `넓은 칼날이 동선을 줄입니다`.
- Rewrite as what the buyer does or feels: `썬 재료를 칼날에 모아 바로 옮길 수 있어요`.

## Spoken Korean Tests

Run these tests on every headline, body paragraph, CTA, FAQ answer, and comparison row.

1. **Kakao test**: Would a real person send this sentence in a KakaoTalk message without sounding strange?
2. **Read-aloud test**: Can it be read aloud in one breath without stumbling?
3. **Seller test**: Would a Korean shop owner say this to a customer in person?
4. **Strategy leak test**: Does it expose a planning term or translated marketing concept?
5. **Skeleton test**: Is the sentence driven by a person/action verb instead of an abstract noun?
6. **Ending test**: Does the final ending match the speaker/listener relationship, section role, and sentence function? If no ending feels natural, rewrite instead of forcing it.

If any test fails, rewrite before scoring.

## Tone Coordinates

Do not define tone only with adjectives such as `단정한`, `실용적인`, `자신감 있는`.

Define tone with:

- speaker persona: `주방용품을 오래 판 사장님이 첫 구매자에게 설명하듯`
- relationship: `처음 사는 사람에게 과장 없이 짚어주는 말투`
- good/bad pair: one sentence that fits, one sentence that must be avoided
- optional reference corpus: if the user supplies strong Korean detail pages or brand copy, mirror their rhythm and word choice

Example:

| Bad | Good |
|---|---|
| 첫 구매의 선택을 줄였습니다 | 처음부터 같이 있으면 따로 찾을 일이 줄어요 |
| 도마 위 동선을 줄입니다 | 썬 재료를 바로 팬으로 옮길 수 있어요 |
| 장비를 쓰는 요리로 전환됩니다 | 좋은 칼 하나 있으면 요리할 맛이 달라져요 |

## Revision Loop

Do not finish after one rewrite. Use this loop:

1. Generate display copy.
2. Extract each visible sentence.
3. Mark failures: strategy leak, English skeleton, abstract subject, noun ending, long breath, tone mismatch.
4. Rewrite failed sentences into spoken Korean.
5. Re-run the five spoken Korean tests.
6. Record the worst examples and rewrites in `COPY_REVIEW.md`.
