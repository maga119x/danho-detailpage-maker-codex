# Consumer Benefit Copy

## Core Rule

Do not sell the product's merit first. Sell the buyer's improved moment.
The buyer, the buyer's situation, or the buyer's desired outcome should be the center of the sentence.
The first visible line should usually start with the buyer's situation, worry, action, or after-state. Add the product feature only as the reason the benefit is believable.

Use this conversion:

`feature -> buyer worry -> daily situation -> buyer action -> benefit -> proof`

For visible ecommerce copy, reorder the final sentence like this:

`daily situation or buyer worry -> buyer action -> benefit -> proof/limit`

Example:

| Feature-first | Buyer-first |
|---|---|
| 네오디뮴 자석으로 강력 고정 | 문에 구멍 내지 않고도 필요한 순간에 문을 고정할 수 있어요 |
| 높이 조절 가능 | 바닥 높이가 달라도 문에 맞춰 사용할 수 있어요 |
| 무타공 설치 | 전세집 현관문에도 부담 없이 붙여 쓰세요 |

## Natural Korean Tone

- Hook: 짧고 직접적으로 말한다. `구매 전, 이 부분만 꼭 확인해 주세요!`
- Pain: 소비자 상황을 먼저 보여준다. `짐 들고 들어올 때 문이 다시 닫히면 불편하니까요.`
- Benefit: 행동과 결과를 함께 쓴다. `발로 살짝 내리면 문이 그대로 고정돼요.`
- Proof: 확인 가능한 사실만 쓴다. `자석이 붙는 철문/방화문에 사용할 수 있어요.`
- FAQ/spec: 정확한 안내형으로 쓴다. `알루미늄문, 목문, 유리문에는 사용할 수 없습니다.`

## Benefit Depth

Each rewrite should answer at least two:

- What annoying moment does this remove?
- What can the buyer do more easily?
- What worry becomes smaller?
- What does the buyer no longer need to do?
- What should the buyer check before buying?

If a section only answers "what is this product?", rewrite again.
If a section only answers "what feature does this product have?", rewrite again.
If the section does not help a mobile shopper decide whether to keep scrolling, check fit, compare, choose, or buy, rewrite again.

## First-Line Rule

The first 1-2 visible lines should carry the purchase meaning. Use one of these:

- Situation first: `짐 들고 들어올 때, 문이 다시 닫히지 않게`
- Fit check first: `구매 전, 자석이 붙는 문인지 확인해 주세요`
- Action first: `붙이고, 내리고, 바로 고정하세요`
- Anxiety first: `전세집이라 구멍이 걱정된다면`
- Option first: `현관문 높이에 맞춰 옵션을 확인해 주세요`

Avoid first lines that only name a feature:

- `강력한 자석`
- `간편한 설치 방식`
- `프리미엄 소재`
- `사용자 편의성`

## Perspective Rule

Seller-centered copy sounds like the brand is the hero. Rewrite it so the buyer's problem, action, or result leads.

| Seller/product-centered | Buyer-centered |
|---|---|
| 저희는 강력한 자석을 사용했습니다 | 문에 구멍 내지 않고도 단단하게 고정할 수 있어요 |
| 이 제품은 편리한 사용성을 제공합니다 | 손이 부족할 때도 발로 간단히 고정할 수 있어요 |
| 브랜드가 전세집 걱정을 해결합니다 | 전세집 현관문에도 자국 걱정을 줄이고 사용할 수 있어요 |
| 사용자를 위해 세 단계 설치를 설계했습니다 | 붙이고, 내리고, 바로 고정하세요 |
| 프리미엄 소재를 담았습니다 | 매일 써도 쉽게 휘청이지 않도록 단단한 소재를 확인해 주세요 |

Allowed product subject:

- factual specs: `본품 1개 구성입니다`
- compatibility: `자석이 붙는 철문/방화문에 사용할 수 있습니다`
- safety/legal limits: `목문, 유리문에는 사용할 수 없습니다`

For selling copy, avoid product-first subjects unless they clarify a fact.

## AI-Slop Phrases To Remove

Avoid these unless the product facts make them unusually true and natural:

- `비밀`, `공개합니다`, `단 하나의`, `당신의 ... 바꿔줄`
- `수많은 분들이 선택한`, `입소문 난`, `검증된` without proof
- `완벽한`, `혁신적인`, `프리미엄 경험`, `삶의 질을 바꿉니다`
- `경험해보세요` as a generic CTA
- Internal labels such as `설치 방식`, `구매 판단 요약`, `불편의 순간`

## Rewrite Patterns

| Weak copy | Better direction |
|---|---|
| 구매 전에는 이 조건만 먼저 확인하세요 | 구매 전, 이 부분만 꼭 확인해 주세요! |
| 자석이 붙는 문이라면 설치 방식은 간단합니다 | 자석이 붙는 문이라면 간단하게 설치할 수 있어요 |
| 세 단계면 충분합니다 | 붙이고, 내리고, 바로 고정하세요 |
| 떼어낸 뒤가 더 중요합니다 | 나중에 떼어도 자국 걱정을 줄였어요 |
| 생활 장면에서 설득됩니다 | 짐 들고 들어올 때, 문이 다시 닫히지 않게 |
| 제품의 강력한 고정력을 제공합니다 | 문이 다시 밀려 닫히지 않도록 필요한 순간에 잡아줘요 |
| 사용자의 편의성을 높였습니다 | 손이 부족할 때도 발로 간단히 고정할 수 있어요 |
| 저희가 꼼꼼하게 만들었습니다 | 매일 쓰는 현관문에 부담 없이 붙여 사용할 수 있어요 |

## Section-Level Guidance

- Hero: one concrete outcome, not a broad slogan.
- Problem: use a familiar moment, not abstract discomfort.
- Solution: state how the buyer uses it.
- Fit check: turn restrictions into a helpful buying checklist.
- Comparison: compare buyer effort, damage risk, and daily convenience.
- Review/proof: include a review/testimonial section. Use real reviews when supplied; otherwise use replacement-ready mock review cards with generic nickname/handle, star rating, highlighted quote, detailed benefit-based copy, and internal replacement notes only.
- Final closing: summarize the product/result impression without purchase-action text.
- Options/value information: make fit, components, and mutable facts understandable without embedding numeric prices, sales-channel names, or final-section action prompts.
- FAQ: answer the worry directly before adding explanation.
